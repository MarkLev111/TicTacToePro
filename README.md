**TicTacToePro — большие и более интересные крестики-нолики с полями 3х3 внутри каждого поля 3х3!**

**Правила:**
1. У каждой большой клетки есть своё маленькое поле внутри;
2. Ставить Х/О можно только в большую клетку, соответствующую маленькой клетке, в которую был поставлен предыдущий Х/О;
3. Первый Х можно поставить в любую клетку;
4. Если кто-то выигрывает большой квадрат или в нём ничья, а следующий ход должен быть совершён в нём, он совершается в любой свободной клетке;
5. Выигрывает тот, кто побеждает в большом поле (то есть 3 нужных маленьких поля).

**Обозначения:**
- зелёный квадрат (маленькая клетка) — можно ставить Х/О;
- белый квадрат (маленькая клетка) — нельзя ставить Х/О;
- красный квадрат (9 маленьких клеток одной большой) — клетка выиграна крестиками;
- синий квадрат (9 маленьких клеток одной большой) — клетка выиграна ноликами.
